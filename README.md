# API_GameCollection

## Descripción del proyecto
API Rest Full .NET para trabajar con base de datos diseñada para mantener colecciones personales de videojuegos físicos.

## Objetivos del proyecto
- Construir API que permita la conexión y manipulación de los datos de la base de datos de colección de videojuegos.

## Stack Tecnológico
- SQL Server
- .NET 10
- Entity Framework
- ASP.NET Core Identity
- Swagger
- AutoMapper
- Git/GitHub
